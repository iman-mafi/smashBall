Game Settings

We will cover the following topics in this section:

1) Setting In-App Purchases

2) Setting Game Center

3) Setting RevMob

4) Changing Game Font

5) Reskinning

 

1) In - App Purchases requires some steps before you can set them up:

- Make sure you have a valid Bundle ID in your iTunes Connect Profile

- Set the same Bundle ID from iTunes Connect to your game -> infoPlist:

- Once  you have set the correct bundle ID try to run the game. If there is no errors you can continue.

Open Constants.h located in your project under helper classes: 

Inside locate the following: 

Change the text "Your IN-App Purchase ID" with the one you created in ITunes Connect for example @"com.envato.removeads"

All done -> Try to remove the ads if you get a message "IN-app coming soon" this means that you have not linked correctly your bundle id or your in-app id is wrong.

 

2) Setting Up Game Center

- First make sure you have connect your bundle id as we did with the in-app purchases. If you already did it move to the next step

- Create in your iTunes Connect a new leaderboard and make sure you activate Game Center. Copy the leaderboard id.

- Open your project and first click on the project and go to capabilities: 

- Make sure Game Center Switch is ON:

- Open again Constants.h

- Find the following: 

- Paste the leaderboard id inside ""

- All done Game center should now work for you

 

3) Setting RevMob 

- In order to set up RevMob you need to create a account and new app in https://www.revmobmobileadnetwork.com

- Once you create the app you will receive a key which you need to copy.

- Open your project and open Constants.h again inside you will find: 

- Change enableRevmob to NO if you dont want to use this network

- In the second field where is my key paste yours

- All done. If you are not seeing a ad your id is wrong

5) Reskinning 

- Open Constants.h and find the following: 

- In order to reskin first you need to drag and drop the new graphics into your project. Once you do that we can start reskinning.

- The first two images are the images for the main character for iphone and ipad. Replace the text of the image with yours and remember to write also the file extension for example: myimage.png

- The second two are the images for the enemy balls.

- The third are the images for the red background (where you touch to move)

- The final image is for the fingerprint which moves around after you run the game (does not require ipad image)
